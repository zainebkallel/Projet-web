# Croissant-Rouge Tunisien - Site Web

## Description
Développement d’un site web centralisé pour le **Croissant-Rouge Tunisien**, visant à connecter les **donateurs** et les **responsables**, avec gestion des dons, des missions et suivi transparent des activités.

Le projet inclut également un **pipeline CI/CD automatisé** pour faciliter le déploiement et le suivi de la qualité du code.

---

## Fonctionnalités principales
- Gestion des dons (ajout, suivi, historique)
- Gestion des missions et des bénévoles
- Suivi transparent des activités
- Interface web réactive pour donateurs et responsables

---

## Technologies utilisées
- **Backend :** Spring Boot  
- **Frontend :** Angular  (vous trouvez le code dans le repository Angular)
- **Base de données :** MySQL
